# Fundamentos de JavaScript

Este proyecto presenta la aplicación de los principales fundamentos de JavaScript mediante ejemplos prácticos en un entorno simulado de red social. Su propósito es reforzar el aprendizaje a través de ejercicios que representan situaciones reales.

---

## Objetivo

Aplicar los conceptos básicos de JavaScript para comprender su funcionamiento en el desarrollo de aplicaciones dinámicas.

---

## Contenido

### Fundamentos
Se trabajó con variables (`var`, `let`, `const`), tipos de datos, estructuras condicionales (`if`, `switch`) y bucles (`for`, `while`, `forEach`) para el control del flujo del programa.

### Objetos
Se implementaron objetos para representar datos estructurados, aplicando desestructuración, uso de métodos y funciones integradas como `Object.keys`, `Object.values` y `Object.assign`.

### Funciones
Se utilizaron diferentes tipos de funciones, incluyendo funciones tradicionales, funciones flecha, funciones anónimas, parámetros por defecto y funciones autoejecutables. También se aplicó el uso de `this` dentro de objetos.

### Arreglos
Se emplearon arreglos para almacenar y manipular datos mediante métodos como `push`, `filter`, `find`, `forEach` y `sort`.

### Manipulación del DOM
Se realizaron operaciones sobre el DOM para crear, modificar y eliminar elementos de una página web, así como la gestión de eventos para permitir interacción del usuario.

### Asincronismo
Se aplicaron conceptos de asincronismo mediante `setTimeout`, `Promise` y `async/await`, permitiendo simular procesos que requieren tiempo sin bloquear la ejecución del programa.

### API y Local Storage
Se trabajó con el consumo de datos externos mediante APIs y el almacenamiento local del navegador utilizando Local Storage para persistencia de información.

### Módulos
Se organizó el código en módulos utilizando `export` e `import`, permitiendo una mejor estructura, reutilización y mantenimiento del código.

---

## Resultados

Se logró integrar múltiples conceptos de JavaScript en un solo proyecto, permitiendo simular el comportamiento básico de una aplicación web. Esto facilitó la comprensión del manejo de datos, la interacción con el usuario y la organización del código.

---

## Conclusión

El desarrollo de este proyecto permitió aplicar de forma práctica los fundamentos de JavaScript, consolidando conocimientos esenciales para la programación web. Además, se evidenció la importancia de estructurar correctamente el código y utilizar herramientas que faciliten su mantenimiento y escalabilidad.
